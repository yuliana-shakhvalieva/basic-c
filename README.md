## basic-c

Домашние задания по основам C++.

#### Задания:
+ Homework_1 - реализация 4 функций: подсчет суммы цифр в переданном числе, подсчет количества простых чисел строго меньше заданного N, целочисленное деление двух чисел a и b, перевод числа из Римской системы;
+ Homework_2 - реализация 4 функций: вычисление следа квадратной матрицы, быстрая сортировка, вычисление уникального элемента в массиве, поиск максимальной площади квадрата состоящего только из `1` в матрице состоящей только из `0` и `1`. 
+ Homework_3 - утилита командной строки для сложения и умножения матриц;
+ Homework_4 - фильтрация изображения с помощью фильтра Гаусса;
+ Homework_5 - сжатие двухпроходным алгоритмом Хаффмана;
+ Homework_6 - реализация класса строки с ленивым копированием.
